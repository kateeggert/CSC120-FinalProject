# COMMANDS

Movement commands to navigate the maze:
 - north 
 - south
 - east 
 - west

Spell commands to cast a spell:
 - Avada Kedavra
 - Expelliarmus
 - Crucio 
 - Sectumsempra 
 - Stupefy

 Other Commands:
 - spell book (prints out spells)
 - hint (prints out a diagram of the maze)
 - end (ends game used for testing purposes)

# LAYOUT DESCRIPTION 

The game is a  maze that leads to the Triwizard Cup. The maze can be accessed at any time by using the appropiate help command. 

# CHALLENGES

The user must solve the riddle (or cast a correct spell) to pass the sphynx. The user then must complete the maze without running out of health. After they fight the maze, Voldemort will appear. They need to cast the correct spells in order to defeat Voldemort.
